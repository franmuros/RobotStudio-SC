Este SC simula la aparición de las obleas de silicio una vez que el prisma ha sido cortado en el segundo almacén cortador. Tiene 7 entradas digitales; cuatro de ellas sirven para producir obleas de silicio del color correspondiente a la simulación que se esté llevando a cabo. Hay una quinta entrada llamada “Velocidad superficie” que tiene por objetivo simular que la cinta es transportadora. Esto se debe a que, mediante el SC de tipo PhysicsControl, se les otorgan movimientos físicos a las obleas para que tengan caída desde el almacén hasta la cinta y puedan deslizar por ella a la velocidad de superficie. Las dos últimas entradas son “Ventosa activa”, que se utiliza para que, mientras esté activa, se pare temporalmente la cinta transportadora, y “Reseteo contador”, usada para resetear el contador del número de obleas producidas. De tal modo, según el color requerido, se activa un SC de tipo Timer, que, cada 2 segundos, activa el SC de tipo Source para crear una nueva oblea que deslizará por la cinta. Por último, mediante puertas lógicas se establece que, cuando una oblea haya llegado al final, se pare la cinta para que sea cogida por el siguiente robot y se reactive al finalizar. Finalmente, se utiliza el SC de tipo Counter para contar las 15 obleas que se soldarán al panel solar.
