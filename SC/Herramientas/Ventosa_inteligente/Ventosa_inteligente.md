Este SC es usado varias veces en la estación en distintos robots para coger las distintas formas en las que se presenta el silicio. Tiene un SC de tipo Line Sensor que marca el objeto a coger. Una vez activada la señal digital de entrada “Vacío”, se activa el SC de tipo Attacher, que vincula el objeto detectado por el sensor lineal con la herramienta que actuará como ventosa. Cuando se desactiva la señal de entrada, se activa el SC de tipo Detacher para soltar el objeto. Por último, tiene una señal digital de salida llamada “vacuostato”, útil para indicar cuándo se está agarrando algún objeto.
