Este SC se encarga de colocar los trozos de pescado en las tablas habilitadas para ello. De manera común, con una entrada lógica que se active cuando se disponga a coger la tabla (Parent) debiendo crear dos SCs de tipo Attacher, uno para cada pieza (Child), acompañado de una puerta lógica para cuando se disponga a dejarla en su zona active el SC de tipo Detacher.
