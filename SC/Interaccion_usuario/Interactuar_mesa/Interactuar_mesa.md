Para el juego de luces preparado, no solo se necesita programación en RAPID. Necesita ir acompañado de una serie de SCs diseñados a tal efecto, que permitan alterar el comportamiento de los botones interactivos de cada mesa, tal y como se presenta. Se tratará de SCs compuestos a partir de SCs básicos de tipo Highlighter y una serie de condicionales que permitan seguir el modelo de las líneas de programación, pero mediante entradas y salidas digitales que permiten esos cambios en el estado del botón interactivo representando a cada una de las mesas y para tener en cuenta cuál está siendo atendida para llevar a cabo la comunicación y cola entre clientes.
