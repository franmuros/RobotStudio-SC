# RobotStudio-SC
Este repositorio contiene componentes inteligentes (SC) compuestos creados mediante el software RobotStudio, que se podrán utilizar libremente por la comunidad académica y profesional. Aquí se podrá:
- Descargar SC encapsulados listos para usar.
- Proponer nuevos SC para ser incluidos en el repositorio. 

Para subir un SC al repositorio se deberán incluir los siguientes documentos:
- Archivo .rslib, que contendrá el SC encapsulado.
- Archivo .md, que contendrá: breve descripción del SC y, en su caso, tipo y función de sus entradas y salidas.

Ambos archivos deberán tener el mismo nombre, el asociado al SC. Si el nombre tiene varias palabras estas se separarán por guiones bajos, y solo estará en mayúsculas la primera letra del archivo. Además, no se usarán caracteres especiales ni acentos en los nombres de archivos. Ejemplo: para un SC denominado "Martillo Neumático", se deberán subir los archivos "Martillo_neumatico.rslib" y "Martillo_neumatico.md".

Para subir los archivos al repositorio, el proceso es el siguiente:
1. Haga un fork de este repositorio.
2. Cree una carpeta dentro de la carpeta "propuestas_pendientes" con el nombre del SC que vaa a subir.
3. Añada a dicha carpeta los dos archivos requeridos.
4. Cree un Pull Request con su propuesta. 

Los administradores del repositorio revisarán su propuesta y, en caso de cumplir las especificaciones, el SC será aceptado como parte del repositorio y movido a la carpeta correspondiente. 
¡Muchas gracias por contribuir!

Este repositorio no acepta ningún tipo de archivo, comentario o contribución que pueda herir la sensibilidad de otras personas, promover discursos de odio, discriminación, violencia o que atente contra los derechos humanos. Se espera que todas las colaboraciones mantengan un tono profesional, respetuoso y constructivo. Las bromas de mal gusto o con contenido ofensivo serán eliminadas y los usuarios responsables podrán ser bloqueados.
