# RobotStudio-SC
Este repositorio contiene componentes inteligentes (SC) compuestas creadas mediante el software RobotStudio. Su descarga y subida está disponible para cualquier interesado de la comunidad académica y profesional.

Para subir una SC al repositorio se deberán incluir los siguientes documentos:
- Archivo .rslib, que contendrá la SC encapsulada.
- Archivo .md, que contendrá:
  - breve descripción de la SC.
  - tipo y función de sus entradas y salidas.
Ambos archivos deberán tener el mismo nombre, el asociado a la SC.
En ningún caso se permitirán SC que atenten 

Ejemplo: Para una SC "Martillo", se deberán subir los archivos "Martillo.rslib" y "Martillo.md".

Los encargados del repositorio revisarán los archivos recibidos y en caso de cumplir las especificaciones la SC será aceptada como parte del repositorio.

Este repositorio no acepta ningún tipo de archivo, comentario o contribución que pueda herir la sensibilidad de otras personas, promover discursos de odio, discriminación, violencia o que atente contra los derechos humanos. Se espera que todas las colaboraciones mantengan un tono profesional, respetuoso y constructivo. Las bromas de mal gusto o contenido ofensivo serán eliminadas y los usuarios responsables podrán ser bloqueados.
