# RobotStudio-SC
Este repositorio contiene componentes inteligentes (SC) compuestas creadas mediante el software RobotStudio, que se podrá utilizar libremente por la comunidad académica y profesional. Aquí se podrá:
- Descargar SC encapsulados listos para usar.
- Proponer nuevos SC para ser incluídos en el repositorio. 

Para subir una SC al repositorio se deberán incluir los siguientes documentos:
- Archivo .rslib, que contendrá la SC encapsulada.
- Archivo .md, que contendrá: breve descripción de la SC, y en su caso, tipo y función de sus entradas y salidas.

Ambos archivos deberán tener el mismo nombre, el asociado a la SC. Si el nombre tiene varias palabras estas se separarán por guiones bajos, y solo estará en mayúsculas la primera letra del archivo. Además, no se usarán caracteres especiales ni acentos en los nombres de archivos. Ejemplo: Para una SC "Martillo Neumático", se deberán subir los archivos "Martillo_neumatico.rslib" y "Martillo_neumatico.md".

Para subir los archivos al repositorio el proceso es el siguiente:
1. Haz un fork de este repositorio.
2. Crea una carpeta dentro de la carpeta "propuestas_pendientes" con el nombre de la SC que vas a subir.
3. Añade a dicha carpeta los dos archivos requeridos.
4. Crea un Pull Request con tu propuesta. 

Los administradores del repositorio revisarán tu propuesta y en caso de cumplir las especificaciones la SC será aceptada como parte del repositorio y movida a la carpeta correspondiente. 
¡Muchas gracias por contribuir!

Se espera que todas las colaboraciones mantengan un tono profesional, respetuoso y constructivo. Las bromas de mal gusto o contenido ofensivo serán eliminadas y los usuarios responsables podrán ser bloqueados.
